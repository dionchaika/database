# Db
The PHP Database Library

## 1. Requirements

1. PHP 7.0
2. PHP PDO extention
3. PHP MySQLi extention

## 2. Installation

Install the PHP database library via [composer](https://getcomposer.org/):

```bash
composer require dionchaika/db:dev-master
```

```php
<?php

require_once 'vendor/autoload.php';
```
